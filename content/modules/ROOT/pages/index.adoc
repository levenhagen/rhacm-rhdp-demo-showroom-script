= Introduction to RHACM Demo - Welcome 👋
:navtitle: Introduction to RHACM Demo

RHACM is a web-based lab instruction platform for the https://demo.redhat.com[Red Hat Demo Platform].
It combines rendered AsciiDoc content with embedded terminals, consoles, and interactive tools -- giving workshop and demo participants everything they need in a single browser tab.
